Install

One line, any machine.

Run the installer, and your browser opens to sign in when it finishes. If your team lead already sent you an install code, see the join page.

macOS & Linux terminal

Paste into any terminal. Installs to ~/.matrix, no sudo, then opens your browser to sign in.

shell 
curl -fsSL https://aiwall.com/downloads/matrix/stable/install.sh | sh

Already have an install code from your team? Append -s -- --token <code> and it onboards without any prompts.

Windows powershell

Paste into PowerShell with the install code from your team page. Installs per-user, no admin rights.

powershell 
irm https://aiwall.com/downloads/matrix/stable/matrix.ps1 | iex; matrix-install -Token <code> -Cloud https://join.visuallogic.ai

Replace <code> with your install code — sign in at the platform and copy it from your Me page.

What talks to what — honestly. Every file on this page is self-hosted on aiwall.com and byte-identical to the Matrix release (verify with SHA256SUMS.txt). Once the installer runs, it connects to the Matrix platform entry (join.visuallogic.ai / matrix.visuallogic.ai) to download the client bundle and complete sign-in — that's the platform itself, not a download mirror — and fetches Node.js from nodejs.org only if your machine doesn't already have a recent one. The mirrored Chinese install guide and the scripts inside the standalone zip likewise reference the platform entry directly; we mirror them unmodified so checksums stay valid.
